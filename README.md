# Number Guessing Game 🎯

## Description
An interactive command-line number guessing game built using Python. Features multiple difficulty levels, scoring system, high score tracking, and real-time feedback.

## Features
- Difficulty levels (Easy, Medium, Hard)
- Smart hints (Too high / Too low / Very close)
- Scoring system based on attempts and time
- High score tracking using JSON
- Replay functionality

## Tech Used
- Python
- JSON (file storage)
- Random module
- Time module

## How to Run
python main.py
